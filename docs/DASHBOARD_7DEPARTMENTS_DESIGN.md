# OZ_A2M 7부서 대시보드 통합 설계

## 개요

README.md에 정의된 7개 부서(관제탑센터, 정보검증분석센터, 보안팀, 유지보수관리센터, 일일 성과분석 대책개선팀, 연구개발팀, 외부 탐색팀)를 대시보드에 통합합니다.

OpenClaw 52개 스킬, pi-mono 7개 패키지, pi-skills 7개 도구, 그리고 다양한 외부 라이브러리를 적극 활용합니다.

---

## 7부서 대시보드 탭 구조

### 제안된 새 탭 구조 (7개 탭)

```
┌─────────────────────────────────────────────────────────────────────┐
│  종합현황  │  데이터수집  │  검증/분석  │  성과분석  │  보안  │  시스템  │  R&D  │
│(Overview) │  (D1/Intel) │   (D2)     │   (D5)    │  (D3) │  (D4)   │  (D6) │
└─────────────────────────────────────────────────────────────────────┘
```

### 기존 탭 vs 새 탭 매핑

| 새 탭 | 기존 탭 | 부서 | 주요 기능 |
|-------|--------|------|----------|
| **종합현황** | overview | 통합 | 총괄 현황, 핵심 지표 |
| **데이터수집** | intel | 제1부서 + 외부탐색 | 실시간 데이터, 수집 상태, Goozi 인텔 |
| **검증/분석** | ai (일부) | 제2부서 | Signal 생성, 검증 파이프라인, AI 분석 |
| **성과분석** | 신규 | 제5부서 | **캘린더 뷰**, 일일 PnL, 리포트 |
| **보안** | 신규 | 제3부서 | 보안 모니터링, Nuclei 스캔, 감사 로그 |
| **시스템** | system | 제4부서 | 시스템 메트릭스, Netdata, Docker |
| **R&D** | stable/dopamine | 제6부서 + 제7부서 | 백테스트, 전략 개발, 봇 제어 |
