## 레전드

- **실선 (───▶)**: 실제 연결됨
- **점선 (- -▶)**: 연결 안됨 / 토픽 불일치
- **❌**: 끊어진 데이터 흐름

## 데이터 저장소

| 저장소 | 용도 | 상태 |
|--------|------|------|
| **Redis** | 캐싱, 상태 관리 | ✅ 연결됨 |
| **MQTT** | 실시간 메시징 | ⚠️ 토픽 불일치 |
| **Elasticsearch** | 로그 저장 | ⚠️ 미연동 |
| **SQLite** | 전략 DB | ✅ D6 사용중 |

## 핵심 발견

**"7개 부서가 모두 떠 있지만, D2→D7→D5→D6→D1 순환 고리가 끊어져 있음"**

- D1은 데이터 수집만 함
- D2는 검증하나 D7이 수신 안함
- D7은 독립적으로 거래만 함
- D5는 거래 결과 못받음
- D6는 분석하나 피드백 안감

**결과: AI 트레이닝 없이 단순 거래 봇 11개만 실행 중**
